# Find-font-tool

 - 查看程序根目录的字体的名称

# Nameid（部分）

| nameID | 含义（中文）                | 含义（英文）                          | 示例                                  |
|--------|-----------------------------|---------------------------------------|---------------------------------------|
| 0      | 版权信息                    | Copyright Notice                      | "Copyright (c) 2020 Adobe Systems Inc." |
| 1      | 字体家族名称                | Font Family Name                      | "Source Han Sans SC"                  |
| 2      | 字体子家族名称（样式）      | Font Subfamily Name                   | "Regular"、"Bold"、"Italic"           |
| 3      | 字体ID（唯一标识）          | Unique Font Identifier                | "SourceHanSansSC-Regular:2019"        |
| 4      | 字体全名（家族+子家族）     | Full Font Name                        | "Source Han Sans SC Regular"          |
| 5      | 字体版本号                  | Version String                        | "Version 2.000"                       |
| 6      | PostScript 名称（关键标识） | PostScript Name                       | "SourceHanSansSC-Regular"             |
| 7      | 商标信息                    | Trademark                             | "Source Han Sans is a trademark of Adobe" |
| 8      | 制造商名称                  | Manufacturer Name                     | "Adobe Systems Incorporated"          |
| 9      | 设计师名称                  | Designer                               | "Adobe Design Team"                   |
| 10     | 描述信息                    | Description                            | "A sans-serif font for Chinese text"  |
| 11     | 供应商名称                  | Vendor URL                             | "http://www.adobe.com"                |
| 12     | 设计师URL                   | Designer URL                           | "http://www.adobe.com/designers"      |
| 13     | 许可证信息                  | License Description                    | "Licensed under the Open Font License" |
| 14     | 许可证URL                   | License URL                            | "http://scripts.sil.org/OFL"          |
